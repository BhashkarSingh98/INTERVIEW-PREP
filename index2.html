1. Difference between flex-row and flex-column?

This is a Css property based on concepts of flex module

In flex-row the items start from the edge of the page it means it works on the main axis from left to right whereas in flex-column the items are aligned from top to bottom.

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

Absolute positioning is positioned relative to the nearest positioned ancestor.we give the position from Top,Bottom,Left , Right these values work for both relative and absolute position.

Q7.What is the difference between padding and margin?

In CSS, a margin is the space around an element's border, while padding is the space between an element's border and the element's content. we can give margin in (-ve ) to overlap the elements.
